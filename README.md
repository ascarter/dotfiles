# dotfiles
My developer workspace profile

## Installation

### Quick Install (Recommended)

Bootstrap a new machine with a single command:

```
sh -c "$(curl -sSL https://raw.githubusercontent.com/ascarter/dotfiles/main/install.sh)"
```

### Manual Install

If you prefer not to pipe to shell, clone the repository first:

```
git clone https://github.com/ascarter/dotfiles.git ~/.local/share/dotfiles
cd ~/.local/share/dotfiles
./install.sh
```

### Uninstall

Remove dotfiles:

```
dotfiles self uninstall
```

Using the script directly:

```
~/.local/share/dotfiles/uninstall.sh
```
